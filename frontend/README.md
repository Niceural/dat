# DAT Front-End

## Instructions

Install dependencies:
```bash
yarn
```

Run dev server:
```bash
yarn dev
```

## Environment variables

```bash
NEXT_PUBLIC_CONTRACT_ADDRESS=<NFT_MINTING_CONTRACT_ADDRESS>
```

## Token data file

Token data lies in `misc/tokenData.json`. Here's an example of what the file looks like:

```json
{
  "tokenId": 1052773
}
```
